## 版本说明

从 v0.0.1 开始，项目采用语义化版本控制：
- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

每个版本都会同时推送到 GitHub 和 Docker Hub，确保代码和镜像的一致性。
